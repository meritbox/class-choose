# 功能需求

## 已完成

### 管理员

1. 登录
2. 教学计划表的修改选课人数，增加删除教学计划，分页
3. 学生表的增加删除修改，分页
4. 班级表的增加删除修改，分页
5. 院系表的增加删除修改，分页
6. 成绩表的查看，分页

### 老师

1. 登录
2. 本学期课程展示，分页

### 学生

1. 登录
2. 选课，进行下列判断
    - 学生的时间是否冲突
    - 学生是否修过这门课
    - 班级容量是否达到上限
3. 退课，选课人数减1
